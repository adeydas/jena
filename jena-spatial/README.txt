Jena Spatial : Simple Spatial Query with SPARQL.
=====================

It's an extension to Apache Jena ARQ, called jena-spatial, which combines SPARQL and simple spatial query. It gives applications the ability to perform simple spatial searches within SPARQL queries. Lucene spatial can be used for the spatial data indexing and searching.

It's a GSoC 2013 project for Apache Jena : http://www.google-melange.com/gsoc/project/google/gsoc2013/jpz6311whu/13003

Home page : http://code.google.com/p/jena-spatial/

User document : http://code.google.com/p/jena-spatial/wiki/JenaSpatialUserDocument
