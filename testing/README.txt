Testing
=======

Test sets can be run with the arq.qtest command line tool (a wrapper round
JUnit).

In the ARQ system test suite:

ARQ/            Tests of the ARQ system
WG-approved/    Tests the working group has approved
WG-pending/     Tests the working group i sthinking about
RDQL/           The original RDQL test suite

Other tests, not usually included in the main 

DAWG/           Working area for tests, usual ones under development
                for the working group

WG-other/       Misc tests from the working group, maybe ones we have
                that illustrate some point or other.

"DAWG" is "Data Access Working Group"
