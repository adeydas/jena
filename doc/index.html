SDB is a component of Jena. It provides for large scale storage and query of RDF datasets using conventional SQL databases. The database tools for load balancing, security, clustering, backup and administration can all be used to manage the installation. SDB is designed specifically to support SPARQL, the query language developed by the W3C RDF Data Access Working Group.

Supported databases include:

  • Microsoft SQL Server 2005 including SQL Server Express
  • Oracle 10gR2 including Oracle Express
  • PostgreSQL v8
  • MySQL 5.0 (>=5.0.22)
  • HSQLDB 1.8
  • Apache Derby 10.2
